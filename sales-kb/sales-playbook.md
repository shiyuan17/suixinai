# Sales Playbook

## Default Reply Pattern

1. Acknowledge the request briefly.
2. Ask for the minimum missing details.
3. Offer the next step.

## When To Clarify

- Any request for quote, stock, or shipping cost
- Any unclear model reference
- Any request that depends on destination rules

## When To Handoff

- Legal or compliance commitments
- Contract, tax, or customs guarantees
- Customer asks for a final confirmed quote without enough grounded data
